ezgif.com news and updates

  • Apr 18, 2023

    Today I added APNG support to the animation speed and overlay tools, and fixed an APNG bug in the censor tool.

  • Dec 5, 2022

    SVG to PNG converter now supports setting the fill color for files that support dynamically changing the color based on currentColor variable (e.g., most icons).

  • Dec 17, 2021

    I made a few users suggested updates to the site:

    • option to manually select files in a specific order for upload to the GIF maker;
    • more FPS options for the video to GIF (apng/webp/avif) converter;
    • opacity parameter for the text tool;
    • support for a few less common video file formats e.g. ts.
  • Nov 22, 2021

    We added dark mode to ezgif.com, it's automatically selected based on your browser settings.
    Upload file size limit for GIFs and other images increased to 50MB.

  • Feb 23, 2021

    ezGIF now supports AVIF image format. You can create, convert, split and edit AVIF files just like GIFs: https://ezgif.com/avif-maker

  • Jan 15, 2021

    A tool for extracting video and image components from Android Motion Picture (MVIMG) files: https://ezgif.com/mvimg-to-video

  • Jan 13, 2021

    Ezgif now has a simple video merge tool, where you can join together multiple videos and GIFs. It supports files in different formats and dimensions. https://ezgif.com/merge-videos

  • Apr 18, 2020

    We added some tools for converting ANI (Animated Windows mouse cursor files) to GIF and other animated image formats.

  • Jan 18, 2020

    We just added one more converter to ezgif - SVG to JPG converter.

  • Sep 27, 2019

    We added a quick and easy barcode image generator with support for various formats and batch processing.
